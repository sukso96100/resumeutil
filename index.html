IntroWorkout

IntroWork 서비스를 이용하여,대학 입시 자기소개서 작성을 연습해 보세요.
웹사이트에서 글자수를 자동으로 세어주어 좀더 편리한 작성 연습이 가능합니다.
인쇄가 필요한 경우, 우측 상단 "메뉴 》 인쇄" 를 사용하면 양식 부분만 인쇄 가능합니다.
"메뉴 》 인쇄" 으로 저장도 가능합니다(저장 기능은 로그인이 필요합니다.)
이 웹사이트는 오픈소스 이며, 여기를 클릭하여 누구든지 웹사이트 개발 및 개선에 참여하실 수 있습니다.
©한영빈(sukso96100@gmail.com)





도움말

간단한 도움말.
  • 이 웹사이트를 이용하시면, 자개소개서 작성 연습을 좀 더 편리하게 하실 수 있습니다.
  • 각 문항마다 내용 입력하셔서 작성 연습을 하시면서, 공백 포함 글자수와 공백 비 포함 글자수를 확인 해 보실 수 있습니다.
  • 작성하신 양식을 인쇄해 보실 수도 있습니다. 우측 상단 메뉴에서 "인쇄" 를 누르면, 양식 부분만 인쇄가 됩니다.
  • 작성중인 양식을 저장하려면, 우측 상단 메뉴에서 "인쇄" 를 누르세요(로그인이 필요합니다.)
  • 현재 이 웹사이트는 자기소개서 공통 문항만 지원합니다.

2015학년도 학생부 전형 자기소개서 공통양식

1. 고등학교 재학기간 중 학업에 기울인 노력과 학습 경험에 대해, 배우고 느낀 점을 중심으로 기술해 주시기 바랍니다(1,000자 이내).

공백 포함 자 입력 됨./ 공백 비 포함 자 입력 됨.

2. 고등학교 재학기간 중 본인이 의미를 두고 노력했던 교내 활동을 배우고 느낀점을 중심으로 3개 이내로 기술해 주시기 바랍니다. 단, 교외 활동 중 학교장의 허락을 받고 참여한 활동은 포함됩니다(1,500자 이내).

공백 포함 자 입력 됨./ 공백 비 포함 자 입력 됨.

3. 학교 생활 중 배려, 나눔, 협력, 갈등 관리 등을 실천한 사례를 들고, 그 과정을 통해 배우고 느낀 점을 기술해 주시기 바랍니다(1,000자 이내).

공백 포함 자 입력 됨./ 공백 비 포함 자 입력 됨.

(4번 질문은 대학교별로 다릅니다, 원하시는 대학교의 질문을 선택하신 후, 답하세요.)

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Rather Drive Stick?
